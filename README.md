# Nevaria University — Security+ Study Hub
A self-contained SY0-701 study portal built as a single HTML file.

## What's inside
- 121 Professor Messer video links with progress tracking
- 18-week study schedule with daily task checkboxes
- 5-domain exam objective checklist
- Practice exam tracker with score calculator
- Key terms / acronym lookup (100+ terms)
- Anki session logger
- Budget tracker
- Study Mode toggle (auto-enables after 6pm)
- Supabase sync for cross-device progress

## Setup
1. Open index.html in any browser — works offline
2. For cross-device sync, fill in Supabase credentials at the top of the script
